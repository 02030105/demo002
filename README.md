# 儿童识字小报生成器

🎨 专为5-9岁儿童设计的智能学习材料生成工具

## 功能特点

- 场景化生成（超市、医院、公园、学校、动物园）
- 自动添加中文和拼音标注
- 简洁易用的Web界面
- 支持图片下载保存

## 使用说明

1. 获取API密钥：访问 https://kie.ai/api-key
2. 输入API密钥
3. 选择场景并输入标题
4. 点击生成小报
5. 等待30-60秒即可生成

## 快速部署到GitHub Pages

1. 在GitHub创建新仓库（公开）
2. 上传此文件夹中的所有文件
3. 进入Settings > Pages
4. Source选择"Deploy from a branch" > "main" > "/ (root)"
5. 保存后即可通过链接访问

## 技术栈

纯HTML/CSS/JavaScript，无需服务器
使用KIE AI Nano Banana Pro模型生成图片
